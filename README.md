# Laboratorio_DMZ
4geeks# Laboratorio: Implementación de una DMZ Segura
Este repositorio contiene la configuración y documentación de un laboratorio de red donde se implementó una Zona Desmilitarizada (DMZ) para asegurar un servidor web.

## Estructura del Proyecto
- `Laboratorio_DMZ.pkt`: Archivo de simulación en Cisco Packet Tracer.
- `/informe`: Contiene el informe técnico detallado.
- `/evidencias`: Capturas de pantalla de las pruebas de conectividad.
